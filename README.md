# NewsAnalyzer

## Running

### Install dependencies

```bash
 pip3 install numpy
 pip3 install tensorflow
```

### Compile weights

Firstly, run
```bash
 python3 main.py
```

### Run project

If you compile weights, next you can run analyzer without training.
To do this, run:
```bash
 python3 start.py
```

### Test
For testing, change the variable called t
```python3
 t = "Наса открыло новую луну Юпитера"
```
